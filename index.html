Cisne Assembly

Created this page to document the basic assembly steps for a cisne keyboard.

Parts

  • Top Case with Magnets x 1
  • Bottom Case with Magnets x 1
  • Plate x 2
  • Hotswap PCB x 2
  • Flat Cable x 1
  • Stabilizer Foam x 8
  • Poron Gasket x 12
  • Plate Foam (between plate and PCB) x 2
  • Thin PE Foam (between the plate foam and PCB) x 2
  • Custom Mold Silicone Rubber Foot x 4
  • Mini Daughterboard x 1
  • JST SH Cable x 1
  • M2 Screw x 1
  • (Optional) Rotary Encoder + Knob

Steps

Step 1

Install the mini daughterboard with the M2 screw. The JST SH cable is used to attach the daughterboard (usb-c connection port) to the keyboard PCB.

Step 2

Attach the poron gaskets to the plates in the protruding portions. Three gaskets are attached to each side of each plate (6 gaskets per plate). The plates are gasket-mounted within the case.

Step 3

Use included holtites to enable the custom layout desired. You'll probably want to solder these for a reliable connection.

Step 4

Install switches using the included foam. PCB on the bottom, then foam, then plate.

Step 5

Attach the ribbon cable. Note that the Sonatina text should be facing down.

Step 6

Place the PCBs in the bottom case and attach the JST SH cable.

Step 7

Install the top case.

Step 8

Finally, install keycaps.

For further information about layouts, mapping, and other cisne details, visit the cisne website.

Reset

To reset the firmware, use tweezers to short the highlighted pads of the right PCB.
